# Asisten AI Sederhana
Asisten AI sederhana mengandalkan respon suara. Dibuat menggunakan bahasa Python.

# Requirements 
Pastikan beberapa library berikut telah terinstall di komputer anda:
- pyttsx3
- Speech Recognition
- PyAudio
- wikipedia
- pipwin
